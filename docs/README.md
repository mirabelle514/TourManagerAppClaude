# Tour Manager App

A comprehensive tour management application for music tours and events.

## Features

- Day Sheet Management
- Financial Tracking
- Merchandise Management
- Team & Crew Management
- Guest List Management
- Communication Tools
- Document Management
- Schedule Management
- Emergency Services
- Roadside Assistance
